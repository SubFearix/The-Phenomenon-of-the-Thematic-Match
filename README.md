# The-Phenomenon-of-the-Thematic-Match

## Парсер результатов матчей ХК «Сибирь» (КХЛ, сезон 2024/25)

Скрипт загружает календарь матчей со страницы
[khl.ru/calendar/1288/00/29/](https://www.khl.ru/calendar/1288/00/29/)
и формирует Excel-отчёт со следующими столбцами:

| Столбец | Описание |
|---|---|
| **Дата** | Дата проведения матча |
| **Соперник** | Название команды-соперника |
| **Дома/В гостях** | `Домашний` / `Гостевой` |
| **Результат** | `Победа` / `Поражение` |
| **Забили** | Количество забитых голов |
| **Пропустили** | Количество пропущенных голов |

### Установка и запуск

```bash
# Создать и активировать виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate   # Linux / macOS
# venv\Scripts\activate    # Windows

# Установить зависимости
pip install -r requirements.txt

# Запустить парсер
python parse_khl.py
```

Результат сохраняется в файл `sibir_results_2024_25.xlsx`.
